Bem-vindo

  • Este jogo é um exercício para memorização do Código Morse
  • O jogo é bem simples, tenha consigo uma tabela de Código Morse em mãos para consulta.
  • Ao pressionar no ícone o jogo começará.
  • Use os botões e para adicionar os pontos e traços.
  • O botão apaga o ultimo caractere em caso de erro.
  • Ao pressionar no ícone o jogo resetará e mostrará os resultados.
  • Ao pressionar no ícone sua resposta é enviada e contada.
  • O objetivo é acertar o máximo de letras e números possíveis.
  • Este site será futuramente atualizado para novas funções.
  • Aproveite para aprender.
